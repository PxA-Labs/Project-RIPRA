# Machine Learning — `rippra/ml/`

PyTorch models for wavefront reconstruction, turbulence prediction, and classification.

## Files

| File | Purpose |
|---|---|
| `models.py` | Model architectures: `WavefrontMLP`, `WavefrontCNN` (ResNet-style 2D conv) |
| `sequence_models.py` | LSTM for prediction + classification: `SequencePredictor`, `SequenceClassifier`, `ParameterEstimator` |
| `train.py` | Training loop for MLP/CNN reconstruction |
| `train_sequence.py` | Training loop for LSTM prediction/classification |
| `evaluate_inference.py` | Compare classical modal vs ML reconstruction, Zernike derivative computation |
| `evaluate_sequence.py` | Evaluate LSTM prediction accuracy, confusion matrix |
| `export_onnx.py` | Export trained models to ONNX format |
| `baseline_comparison.py` | Phase 8.1 — RMSE/correlation/Strehl comparison |
| `noise_robustness.py` | Phase 8.2 — Gaussian/photon/occlusion sweeps |
| `ablation_study.py` | Phase 8.3 — MLP width/depth, LSTM lookback |
| `performance_profile.py` | Phase 8.4 — 500-iteration latency profiling |
| `benchmark_gpu.py` | GPU vs CPU throughput benchmark |

## Models

| Model | Params | Test MSE | Latency | Description |
|---|---|---|---|---|
| MLP (2-layer FC) | 35K | 0.752 | 0.67 ms | Baseline fully-connected |
| CNN (ResNet) | 206K | **0.0106** | 2.26 ms | 2D conv on 12×12 physical grid |
| LSTM (predictor) | 211K | RMSE 1.72 | 0.62 ms | 10-frame lookahead |
| LSTM (classifier) | 211K | 99.64% acc | 0.62 ms | Weak/Moderate/Strong |

## Dataset

Training data generated by `tools/generate_dataset.py`:
- Kolmogorov turbulence with Taylor frozen-flow AR(1) evolution
- Noll covariance matrix for spatial correlations
- 10,000 samples, D/r₀ ∈ [1, 10], τ₀ ∈ [2, 10] ms

## Usage

```bash
# Train models
python train.py
python train_sequence.py

# Evaluate
python evaluate_inference.py
python evaluate_sequence.py

# Export to ONNX
python export_onnx.py

# Run analysis
python baseline_comparison.py
python noise_robustness.py
python ablation_study.py
python performance_profile.py
```

## One-Command Reproduction

The entire ML pipeline (C calibration → dataset generation → model training → ONNX export → validation) can be reproduced from scratch with a single command:

```bash
# From the rippra/ directory:
python tools/reproduce_all.py
```

This will:
1. Build the C library and run `test_centroid` to generate reference centroids
2. Generate a 500-sample Kolmogorov turbulence dataset
3. Train MLP and CNN reconstructors (3 epochs each — fast config)
4. Export both models to ONNX format
5. Validate ONNX model input/output shapes
6. Run predictive AO simulation

**CI:** This pipeline runs automatically on every push via the `Python ML Reproducibility` job.

**Docker:** The published Docker image also runs the full pipeline and ships the trained ONNX models.
